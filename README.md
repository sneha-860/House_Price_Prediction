# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using Machine Learning techniques. It uses a dataset from Kaggle and applies regression models to estimate property prices based on features.

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## 📊 ML Model

* Linear Regression

## 📁 Dataset

* Source: Kaggle (House Prices Dataset)

## 🚀 Steps Performed

1. Data Cleaning
2. Feature Selection
3. Model Training
4. Evaluation

## 📈 Results

* R2 Score: 0.8227439852048932

## 🔮 Future Scope

* Improve accuracy using Random Forest / XGBoost
* Build web app using Streamlit
* Deploy model online
